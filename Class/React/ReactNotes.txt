# React
-React is a js library , developed by meta
-It is single page application.

#why do we use React
-traditional js reloads full page
-hard to manage large application 
-But React upadates only required part , use reusable components

# Virtual DOM
-React creates a copy of real DOM means virtual DOM.
-when data changes react updates virtual DOM.
-updates only changed part in real DOM.
-used for High Performance.
